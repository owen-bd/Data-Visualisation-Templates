# DataVisualisationTemplates
A Collection of Templates for Data Visualisation:

* Bar Charts  
* Dot Plots  
* Histograms  
* Joy Plots  
* Line Plots  
* Lollipop Charts  
* Radar Plots  
* Scatterplots  
* Swarm Plots  
* Treemaps  
* Violin Plots   
<br>
These do not include templates or functions for data cleaning or transformation, as it assumed this will be done prior.
